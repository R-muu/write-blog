## 📌 3. 記事の書き方ルール

### 🖋 文体・トーン
- 敬語とタメ語を自然に混ぜる（親しみやすい口調）
- 絵文字を適度に（1段落に1〜2個）
- 色文字・強調を効果的に使う
- 難しい専門用語は使わない

### 📐 構成
```
導入1段落（共感・問いかけ）
  ↓
ポイント2〜3つ（見出し付き）
  ↓
まとめ1段落（「続きは講座で！」への誘導）
```

### 🎯 コンテンツ原則
- **「もっと知りたい！」と思わせる**書き方
- 核心的なHow toは書かない → **「続きは講座で」**
- 読者が**今日からできる小さな一歩**を1つ提示

## 📌 6. SNS連携戦略

### 🔗 連携フロー
```
ブログ記事公開
   ↓
┌─────────────┬─────────────┐
│  Instagram   │   Threads    │
│ （画像＋長文） │ （短文＋会話） │
└─────────────┴─────────────┘
   ↓
すべて「プロフリンク → ブログ」へ誘導
```

### 📝 プラットフォーム別の書き方

| | Instagram | Threads |
|--|-----------|---------|
| **主役** | 写真・画像 | テキスト |
| **文字数** | 200字前後 | 50〜100字 |
| **文体** | キャプション的 | 会話的・ひとこと |
| **ハッシュタグ** | 20〜25個 | 1〜2個 |
| **誘導先** | プロフリンク→ブログ | 文末リンク→ブログ |

### 🔄 自動/半自動連携の選択肢
| 方法 | 内容 | 推奨度 |
|------|------|--------|
| **Meta Business Suite** | Instagram→Threads同時投稿（無料） | ⭐⭐⭐ |
| **IFTTT/Make** | ブログ更新→SNS通知（無料〜） | ⭐⭐ |
| **手動コピペ** | 毎回微調整できる（確実） | ⭐⭐⭐ |

**推奨：** 手動運用でOK。記事セット7点が揃っているので、コピペですぐ投稿可能。

## 📌 8. 書いていいこと・ダメなこと

### ✅ OK
- 健康な生理の基本的な特徴（8か条の**概要**）
- 冷え・骨盤・睡眠・食事と生理の関係（一般的な健康情報）
- 骨盤底筋トレーニングの**概念・目的**
- 自分の体験談・気づき
- 「続きは講座で！」への誘導

### ❌ NG
| カテゴリー | 禁止内容 |
|----------|---------|
| 📜 **著作権** | 協会テキスト・資料の文章をそのまま引用／転載 |
| 💊 **医療情報** | ピル・IUD・HRTの詳細説明／否定的記述 |
| 🏥 **疾患断定** | 「内膜症は〇〇が原因」など断定的な疾患説明 |
| 🧪 **根拠曖昧** | 経皮毒の主張、「昔は自然にトイレで」の歴史主張 |
| 🎓 **講座の核心** | 経血コントロールの具体的Howto |
| ⚠️ **危険情報** | 排尿を途中で止めるトレーニング（膀胱炎リスク） |
| ®️ **商標** | 協会ロゴの無断改変使用 |
